## 📋 Sobre o Projeto
Sistema desktop para gerenciamento de solicitações de vale transporte (adesão, renúncia e alteração) desenvolvido em Java com SQLite.

## 🎯 Objetivo
Centralizar e facilitar a gestão de solicitações de vale transporte, permitindo:
- Cadastro manual de solicitações
- Importação de planilhas Excel
- Consultas por nome, CPF ou matrícula
- Geração de relatórios com gráficos
- Exportação de dados

## 🛠️ Tecnologias
- Java 17+
- SQLite3
- Java Swing (Interface)
- Apache POI (Excel)
- JFreeChart (Gráficos)
